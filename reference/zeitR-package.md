# zeitR: Actigraphy Data Parsing and Analysis for R

Provides tools for importing, parsing, and analysing raw actigraphy
data. Supports common device file formats, computes standard
rest-activity variables (IS, IV, RA, L5, M10), and assembles tidy data
frames ready for downstream chronobiological analysis. Designed to
complement `slumbR` in the Circadia Lab ecosystem.
