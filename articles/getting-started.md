# Getting started with zeitR

## Overview

**zeitR** is an R package for importing, parsing, and analysing raw
actigraphy data. It is designed to complement `slumbR` in the Circadia
Lab ecosystem and provides a tidy, pipeline-friendly interface for
common rest-activity analyses.

## Installation

``` r

# Install from GitHub (development version)
# install.packages("pak")
pak::pak("circadia-bio/zeitR")
```
